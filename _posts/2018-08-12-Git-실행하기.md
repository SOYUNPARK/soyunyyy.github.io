## Git 실행
Git 설치 후 [시작메뉴] - [Git] - 'Git Bash' 실행한다.  

___
<br>
## 사용자 정보 입력  
사용자 정보를 입력하여 권한이 있는 저장소에 접근을 하도록 한다.

`git config --global user.name "사용자이름"`
`git config --global user.email "이메일"`

사용자 정보를 입력하였다면, 아래의 명령어로 확인이 가능하다.

`git config --global -l`
